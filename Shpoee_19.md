# shopee

2019年还剩15天就要结束了，想想今年我也没做啥特别的，值得纪念的一点就是换了工作。今年5月底我离开工作了三年的金蝶，加入了shopee。这次换工作没有面特别多，只面了腾讯，商汤，shopee和vivo，这次的诉求比较简单，拿到让我满意的薪资，最终也如愿以偿。
 一眨眼入职shopee就半年了，接下来我要如实的说下我在shopee工作体验，包括好的和有待改进的地方。

#### 特点：

1. 公司扩张挺快的，5月份我入职的时候深圳团队大概是700多人，现在有1200多人了
2. 公司不提倡加班，给的薪水也算厚道。所以能招到挺多大厂出来的员工。经常能看到腾讯美团京东饿了么等等厂友面基
3. 相对比较年轻，内部目前也没有严格的职级划分以及晋升通道。例如研发来说，只分为：工程师，高级工程师，管理者等

#### 优点：

1. 内部氛围很好，同事们都很友善，相处的非常愉快。我们小组的人都非常的有趣，若不是他们都有了女朋友，hiahiahia...那我可就不客气了...
2. 由于深圳是研发中心，所以对研发质量和技术比较重视。内部利用全流程，三段式等方式来管理研发过程，对于文档以及技术分享也相对重视（相比于我以前）。
3. 公司鼓励大家学习，每月都会有技术类和非技术类的课程，每月也都有极客时间等的免费课程可以报。

#### 缺点：

1. 由于深圳是研发中心，很多核心的一些数据还在新加坡，再涉及到跨国协作。研发起来会有诸多不便，系统设计也变得复杂了很多，接口间会有各种封装和调来调去。
2. 产品经理分为商业产品经理和功能性产品经理。商业产品经理主要负责市场和用户，功能性产品经理负责研发。商业产品经理主要在新加坡，深圳主要是功能性产品经理。有时候一个功能是由商业产品经理提出，即使大家觉得没有必要或者不那么合理，可能也无法改变，所以深圳产品经理们的工作一定程度会受限。
3. 研发流程比较长（产品经理设计需求出prd  + 开发开发 + 测试 + local当地人测试（该流程比较长）+ 回归测试 + 上线），所以有时候会觉得需求迭代比较慢。
4. 目前有些团队QPS不算大，日常无法遇到高并发场景的挑战。

#### 总结:

总体来说 ，shopee还是不错的，推荐大家入坑
